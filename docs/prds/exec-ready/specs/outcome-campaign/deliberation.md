## Recommendation

Folding all three CONTESTED verdicts plus the two HOLDS conditions, the recommendation lands as the 8-point anchor with added run-safety, oracle-absence, negative-space, and re-entry constraints:

1. **Skeleton + oracle + capability (C1):** thin-orchestrator campaign SKILL mirroring review-board Step 0–6 (C1a1); oracle-by-ID = in-scope FR-018 outcome ACs + declared money/safety rules, NOT re-derived (C1c1); capability-detect entrypoint from `.spec-flow.yaml campaign.entrypoint` (absent ⇒ SKIPPED, not error — backward-compat d) + `--run` override (C1b1/b3); SKIPPED-per-stage never-false-green (C1d1). System runs on **Sonnet**, grading on **Opus** (C1e1, NN-P-005).
